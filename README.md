# 🌐 Docs-For-Wikipedia

A high-performance, minimalist documentation engine styled after [Murinopedia](https://pap3ryyy.github.io). This project provides a centralized resource for understanding **Wikipedia's** internal architecture, wikitext syntax, and API integration.

## 🚀 Key Features

- **🎯 Specialized Content**: Deep dive into MediaWiki's core (PHP/MySQL) and page lifecycle.
- **⚡ Ultra-Fast Navigation**: Built as a Single Page Application (SPA) with zero-latency section switching.
- **🌓 Modern Nord Aesthetics**: Optimized for developers with a deep-dark Nord color palette.
- **🌍 Advanced Localization**: Instant toggling between **Ukrainian** and **English** with persistent state.
- **🛠 Code-Ready Examples**: Syntactically highlighted snippets for Wikitext and API GET requests.
- **📱 Fully Responsive**: Seamless reading experience across Desktop, Tablet, and Mobile.

---

## 🛠 Tech Stack

This project is built using a "Vanilla First" approach for maximum speed and zero dependencies:
- **HTML5**: Semantic structure for SEO and accessibility.
- **CSS3**: Advanced layouts using Flexbox and CSS Variables (Custom Properties).
- **JavaScript (ES6+)**: Custom rendering engine with `localStorage` persistence.
